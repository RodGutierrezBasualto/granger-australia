MARA Code of Conduct

The Migration Agents Registration Authority Code of Conduct is intended to regulate the conduct of registered migration agents. It is to be displayed prominently in a registered migration agent's office and reception area. It is also to be given to a client to keep if he or she asks to see it.

Overview

The Code details the professional and ethical standards for registered migration agents. This Code is legally provided for in section 314 of the Migration Act 1958, and it is prescribed in Schedule 2 of Regulation 8 of the Migration Agents Regulations 1998.

Professional Standards: All registered migration agents must adhere to this Code to ensure ethical practice and protection of clients.

Key Principles

  • Act with integrity and professionalism
  • Provide accurate and honest advice
  • Avoid conflicts of interest
  • Maintain client confidentiality
  • Comply with all immigration laws and regulations
  • Keep proper records of client matters
  • Handle client funds appropriately

Further Information

For complete details about the Code of Conduct and what to expect from your migration agent, please visit the Migration Agents Registration Authority website.
